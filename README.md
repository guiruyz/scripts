# Scripts
## Bash scripts
### There's some files into this repositories, with log capture proposes
#### start iperf-logs
```
./iperf-logs.sh <ip> <porta> <banda> <arquivo_log>
```
#### cpu-logs
##### capture only one cpu
```
./cpuId-logs.sh <arquivo_log> <cpuid>
```
##### capture all cpus
```
./cpu-logs.sh <arquivo_log>
```
